# route-test

这个项目实现了：
	-/（\.\*) 路径下的网页会显示 一个按钮，按下后会跳转到 /$1\1 路径下

	-在发布这个项目的时候，会预先生成 /114514 /1919810 两个路径时的网页渲染结果

需要进一步尝试的：
	-将发布后的网页app放到一个asp.net core 后端上，测试预渲染的相性
